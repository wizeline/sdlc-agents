# Confluence Security Report — Page Template

Use this template to structure any security artifact as a Confluence page.
Select the section layout matching the artifact type.

---

## Universal Page Header (all artifact types)

```markdown
> **Security Agent** | {Artifact Type} | Generated: {YYYY-MM-DD}
>
> **Scope:** {Application / Service / System name}
> **Assessed by:** Security Agent (AI-assisted) — validate findings before acting
> **Frameworks referenced:** {e.g., OWASP Top 10 2025, ASVS 5.0, CWE Top 25, NIST SSDF 1.1}
> **Audience:** {Technical team / Engineering leadership / Auditors}
```
